# Cozy

Cozy is a simple application that adjusts the brightness of your screen to match the ambient light in your room.

## Features

- Adjust brightness based on the ambient light in your room
- Supports multiple monitors
- Supports multiple screens
- Supports multiple displays
- Supports multiple screens per display
- Supports multiple displays per screen
- Supports multiple screens per display per monitor
- Supports multiple displays per screen per monitor
- Supports multiple screens per display per monitor per desktop
- Supports multiple displays per screen per monitor per desktop

## Installation

### Prerequisites

Before installing Cozy, make sure you have the following prerequisites installed:

- [Python 3.12](https://www.python.org/downloads/) or later
- [pip](https://pip.pypa.io/en/stable/installation/)

### Run Cozy

To run Cozy, run the following command:

```bash
git clone https://github.com/yagiz/cozy.git
cd cozy
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```
